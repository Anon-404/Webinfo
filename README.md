## Installation

For debian based linux os

```bash
sudo apt install whois -y
sudo apt install host -y
git clone https://github.com/Anon-404/Webinfo/
cd Webinfo
chmod 777 *
bash webinfo.sh
```
For termux

```bash
pkg install whois -y
pkg install git -y
pkg install host -y
git clone https://github.com/Anon-404/Webinfo/
cd Webinfo
chmod 777 *
bash webinfo.sh
```

For arch based linux os

```bash
sudo pacman -Sy whois -y
sudo pacman -Sy host -y
git clone https://github.com/Anon-404/Webinfo/
cd Webinfo
chmod 777 *
bash webinfo.sh
```
